Running Astro hybrid on Workers

Most portfolios ship as a single static bundle. This one runs as three independently deployable Cloudflare Workers, and the win was bigger than I expected.

The split

The site worker renders Astro in hybrid mode. Static pages prerender; dynamic routes server-render at the edge.

$ wrangler deploy --name infra-folio

[✓] published https://phuctruong.dev
