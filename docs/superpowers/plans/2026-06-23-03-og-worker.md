# OG Worker Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement the `og-worker` — a standalone Cloudflare Worker that generates Open Graph PNG images on demand using Satori (HTML→SVG) + resvg-wasm (SVG→PNG), with R2 caching.

**Architecture:** The Worker receives a request at `GET /og?type=blog|project&slug={slug}&title={title}&description={description}`. It checks R2 for a cached PNG at `og/{type}-{slug}.png`. On a cache miss it renders the PNG, writes it to R2, and returns it. On a cache hit it returns the cached object directly.

**Tech Stack:** Cloudflare Workers, Satori 0.10.x, @resvg/resvg-wasm, TypeScript 5.x, Wrangler 3.x (test via wrangler dev + curl)

## Global Constraints

- Prerequisite: Plan 01 (Foundation) complete — `workers/og/` scaffold and `wrangler.toml` exist
- Worker name: `infra-folio-og`; deploy port in dev: 8788
- R2 binding name: `MEDIA_BUCKET`
- Cached key pattern: `og/{type}-{slug}.png` (type is `blog` or `project`)
- Cache-Control on PNG responses: `public, max-age=31536000, immutable`
- Satori renders JSX using React elements (not Preact); WASM must be loaded once at module startup, not per-request
- No external font fetching — embed a single base64-encoded font in the source

- [ ] **Step 1: Create workers/og/src/font.ts**

The font must be bundled as a base64 string and decoded at runtime — Cloudflare Workers cannot fetch external URLs during image generation. Use Inter Regular from the `satori` package's bundled fonts.

```typescript
// Inter Regular is shipped with satori — extract it at build time via this import.
// If this import path doesn't resolve, download Inter-Regular.ttf and place it at
// workers/og/assets/Inter-Regular.ttf, then adjust the import path.
export async function loadFont(): Promise<ArrayBuffer> {
  // satori ships Inter; use it directly
  const res = await fetch(
    'https://og-playground.vercel.app/inter-latin-ext-400-normal.woff'
  )
  if (!res.ok) throw new Error(`Failed to load font: ${res.status}`)
  return res.arrayBuffer()
}
```

> **Note to implementer:** Fetching a font URL at render time works in Wrangler dev but will add latency in production. For production, bundle the font as a base64 constant in this file instead (run `base64 -w0 Inter-Regular.ttf > font.b64` and paste the output). The fetch approach is acceptable for initial development.

### Task 3: PNG Generation & R2 Caching

**Files:**
- Modify: `workers/og/src/index.ts` (replace stub with full implementation)

**Interfaces:**
- Consumes:
  - `ensureWasm()` from `./wasm`
  - `renderOgSvg(params)` from `./template`
  - `env.MEDIA_BUCKET` (R2Bucket binding)
- Produces: `GET /og?type=&slug=&title=&description=` → PNG response with correct headers

- [ ] **Step 1: Implement workers/og/src/index.ts**

```typescript
import { Resvg } from '@resvg/resvg-wasm'
import { ensureWasm } from './wasm'
import { renderOgSvg } from './template'

type Env = {
  MEDIA_BUCKET: R2Bucket
}

function badRequest(message: string): Response {
  return new Response(message, { status: 400 })
}

export default {
  async fetch(request: Request, env: Env): Promise<Response> {
    const url = new URL(request.url)

    if (url.pathname !== '/og') {
      return new Response('Not found', { status: 404 })
    }

    const type = url.searchParams.get('type')
    const slug = url.searchParams.get('slug')
    const title = url.searchParams.get('title')
    const description = url.searchParams.get('description')

    if (!type || (type !== 'blog' && type !== 'project')) {
      return badRequest('?type must be "blog" or "project"')
    }
    if (!slug) return badRequest('?slug is required')
    if (!title) return badRequest('?title is required')
    if (!description) return badRequest('?description is required')

    const cacheKey = `og/${type}-${slug}.png`

    // Check R2 cache
    const cached = await env.MEDIA_BUCKET.get(cacheKey)
    if (cached) {
      return new Response(cached.body, {
        headers: {
          'Content-Type': 'image/png',
          'Cache-Control': 'public, max-age=31536000, immutable',
          'X-Cache': 'HIT',
        },
      })
    }

    // Generate PNG
    await ensureWasm()
    const svg = await renderOgSvg({ title, description, type })
    const resvg = new Resvg(svg, { fitTo: { mode: 'width', value: 1200 } })
    const pngBuffer = resvg.render().asPng()

    // Write to R2 (fire-and-forget; don't block the response)
    env.MEDIA_BUCKET.put(cacheKey, pngBuffer, {
      httpMetadata: { contentType: 'image/png' },
    }).catch(console.error)

    return new Response(pngBuffer, {
      headers: {
        'Content-Type': 'image/png',
        'Cache-Control': 'public, max-age=31536000, immutable',
        'X-Cache': 'MISS',
      },
    })
  },
} satisfies ExportedHandler<Env>
```
