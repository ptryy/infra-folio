# API Worker Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement the `api-worker` with three endpoints — `/status` (real-time), `/github/feed` (KV-cached), and `/github/stats` (KV-cached) — proxying the GitHub API through the Worker so `GITHUB_TOKEN` is never exposed to browsers.

**Architecture:** Single-file Worker with a URL-based router. KV (`GITHUB_CACHE`) stores serialized JSON with a 5-minute TTL. The Worker sets permissive CORS headers so the `GithubFeed` Astro island can call it from any origin during development.

**Tech Stack:** Cloudflare Workers, Workers KV, TypeScript 5.x, Wrangler 3.x

## Global Constraints

- Prerequisite: Plan 01 (Foundation) complete — `workers/api/` scaffold and `wrangler.toml` exist
- Worker name: `infra-folio-api`; dev port: 8787
- KV binding: `GITHUB_CACHE`
- GitHub token binding: `GITHUB_TOKEN` (wrangler secret in prod, set as `[vars]` in local wrangler.toml for dev)
- KV cache TTL: 5 minutes (300 seconds)
- `/status` is NOT cached — returns real-time data
- CORS: `Access-Control-Allow-Origin: *` on all responses (the Worker is the public API, not a private service)

```typescript
type Env = {
  GITHUB_CACHE: KVNamespace
  GITHUB_TOKEN: string
}

const CORS_HEADERS = {
  'Access-Control-Allow-Origin': '*',
  'Access-Control-Allow-Methods': 'GET, OPTIONS',
  'Access-Control-Allow-Headers': 'Content-Type',
}

function json(data: unknown, status = 200): Response {
  return new Response(JSON.stringify(data), {
    status,
    headers: { 'Content-Type': 'application/json', ...CORS_HEADERS },
  })
}

function notFound(): Response {
  return json({ error: 'Not found' }, 404)
}

export default {
  async fetch(request: Request, env: Env): Promise<Response> {
    if (request.method === 'OPTIONS') {
      return new Response(null, { status: 204, headers: CORS_HEADERS })
    }

    const url = new URL(request.url)

    if (url.pathname === '/status') return handleStatus()
    if (url.pathname === '/github/feed') return handleGithubFeed(env)
    if (url.pathname === '/github/stats') return handleGithubStats(env)

    return notFound()
  },
} satisfies ExportedHandler<Env>

async function handleStatus(): Promise<Response> {
  return json({ status: 'ok' })
}

async function handleGithubFeed(_env: Env): Promise<Response> {
  return json({ events: [] })
}

async function handleGithubStats(_env: Env): Promise<Response> {
  return json({ stars: 0, commits: 0 })
}
```

### Task 2: KV Cache Helper

**Files:**
- Create: `workers/api/src/kv-cache.ts`

**Interfaces:**
- Produces:
  - `kvGet<T>(kv, key) → Promise<T | null>` — deserializes JSON from KV; returns null on miss
  - `kvSet(kv, key, value, ttlSeconds) → Promise<void>` — serializes and stores with TTL

- [ ] **Step 1: Create workers/api/src/kv-cache.ts**

```typescript
export async function kvGet<T>(kv: KVNamespace, key: string): Promise<T | null> {
  const raw = await kv.get(key)
  if (!raw) return null
  try {
    return JSON.parse(raw) as T
  } catch {
    return null
  }
}

export async function kvSet(
  kv: KVNamespace,
  key: string,
  value: unknown,
  ttlSeconds: number
): Promise<void> {
  await kv.put(key, JSON.stringify(value), { expirationTtl: ttlSeconds })
}
```

- [ ] **Step 2: Replace handleGithubFeed stub**

```typescript
async function handleGithubFeed(env: Env): Promise<Response> {
  const CACHE_KEY = `github:feed:${GITHUB_USERNAME}`

  const cached = await kvGet<unknown[]>(env.GITHUB_CACHE, CACHE_KEY)
  if (cached) {
    return json({ events: cached, cached: true })
  }

  const res = await fetch(
    `https://api.github.com/users/${GITHUB_USERNAME}/events/public?per_page=10`,
    {
      headers: {
        Authorization: `Bearer ${env.GITHUB_TOKEN}`,
        'User-Agent': 'infra-folio-api/1.0',
        Accept: 'application/vnd.github.v3+json',
      },
    }
  )

  if (!res.ok) {
    return json({ error: `GitHub API error: ${res.status}` }, 502)
  }

  const events = await res.json() as unknown[]

  await kvSet(env.GITHUB_CACHE, CACHE_KEY, events, GITHUB_CACHE_TTL)

  return json({ events, cached: false })
}
```

**Files:**

The GitHub API endpoint used: `GET https://api.github.com/users/{username}`

- [ ] **Step 1: Replace handleGithubStats stub**

```typescript
async function handleGithubStats(env: Env): Promise<Response> {
  const CACHE_KEY = `github:stats:${GITHUB_USERNAME}`

  const cached = await kvGet<{ stars: number; repos: number; followers: number }>(
    env.GITHUB_CACHE,
    CACHE_KEY
  )
  if (cached) {
    return json({ ...cached, cached: true })
  }

  const userRes = await fetch(`https://api.github.com/users/${GITHUB_USERNAME}`, {
    headers: {
      Authorization: `Bearer ${env.GITHUB_TOKEN}`,
      'User-Agent': 'infra-folio-api/1.0',
      Accept: 'application/vnd.github.v3+json',
    },
  })

  if (!userRes.ok) {
    return json({ error: `GitHub API error: ${userRes.status}` }, 502)
  }

  const user = await userRes.json() as {
    public_repos: number
    followers: number
    public_gists: number
  }

  // Fetch total stars by summing across repos (first page, up to 100 repos)
  const reposRes = await fetch(
    `https://api.github.com/users/${GITHUB_USERNAME}/repos?per_page=100&sort=pushed`,
    {
      headers: {
        Authorization: `Bearer ${env.GITHUB_TOKEN}`,
        'User-Agent': 'infra-folio-api/1.0',
        Accept: 'application/vnd.github.v3+json',
      },
    }
  )

  let stars = 0
  if (reposRes.ok) {
    const repos = await reposRes.json() as Array<{ stargazers_count: number }>
    stars = repos.reduce((sum, r) => sum + r.stargazers_count, 0)
  }

  const stats = { stars, repos: user.public_repos, followers: user.followers }
  await kvSet(env.GITHUB_CACHE, CACHE_KEY, stats, GITHUB_CACHE_TTL)

  return json({ ...stats, cached: false })
}
```
